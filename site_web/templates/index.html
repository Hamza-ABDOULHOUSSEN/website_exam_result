Vous cherchez une information sur un élève ou sur certains résultats ? Ce site vous permettra de consulter aisément les données du concours afin de trouver les informations que vous cherchez rapidement.

Objectifs

Ce site sert à consulter les données du concours Mines-Telecom. Nous avons gardé une interface simple afin que l'utilisateur prenne en main la navigation sur le site rapidement. Nous avons deux onglets de recherche et un de consultation. Le premier onglet de recherche permet de faire une requête sur un candidat. Le second onglet de recherche permet de trouver des statistiques sur le concours. L'onglet de consultation est simplement un onglet qui permet d'obtenir les nombres d'admis et d'admissible par filière.
