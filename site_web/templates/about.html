A propos de ce site

Ce site a été réalisé dans le cadre d'un projet pour Télécom Nancy en 2021. Nous sommes un groupe de 3 étudiants, Lucas Gouin, Hamza Abdoulhoussen et Benjamin Lavandier. Nous devions créer une base de données pour stocker les informations du concours puis réaliser une application pour les consulter. D'où la conception de ce site web pour accéder aux informations des candidats ou des résultats du concours.
